TC ID: TC001  
Module: User Login  
Test Case Description: Verify that a user can successfully log in with valid credentials.  
Preconditions: User must have a registered account.  
Test Steps: 1. Navigate to the login page. 2. Enter valid username. 3. Enter valid password. 4. Click on the 'Login' button.  
Expected Result: User is successfully logged in and redirected to the dashboard.  
Priority: High  
Test Type: Positive Testing
